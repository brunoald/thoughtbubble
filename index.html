Reflex

Reflex is a front-end development framework written in HTML5, JavaScript and Sass. It is designed to serve as a starting point for modern web projects and has been developed from the codebase we have been developing responsive websites with at Doublethink with for around two years now. Due to it's design studio roots Reflex does not attempt to restrict the style of the site with it's default settings, its goal is to combine sensible defaults with robust grid building tools

Browser Support

Reflex is designed to work fully with the latest release versions of Chrome, Firefox, Safari, Opera and Internet Explorer 8+, as well as modern smartphone browsers (e.g. Android and iOS) with the basic (non-media query) styles serving as a base for supporting older browsers.
